# App Name Suggestions

Brandable, English names for the AAC/ASK picture-card generator. All names incorporate **ASK** or **AAC** as a recognisable component.

---

## ASK-based names

| # | Name | Rationale |
|---|------|-----------|
| 1 | **ASKcard** | Straightforward compound — ASK + card. Clear, descriptive, easy to search. |
| 2 | **ASKdeck** | A *deck* of cards implies a complete, ready-to-use set. Short and punchy. |
| 3 | **ASKcraft** | Positions the tool as something you *craft* with care, not just generate. |
| 4 | **ASKstudio** | Suggests a full creative workspace; works well for the GUI app. |
| 5 | **ASKpress** | Press = print + publish. Clean and professional. |
| 6 | **ASKforge** | "Forging" cards conveys strength and precision. Memorable. |
| 7 | **ASKkit** | A *kit* you assemble and hand to someone — fits the print-laminate-cut workflow perfectly. |
| 8 | **ASKflow** | Emphasises the smooth workflow from images to finished PDF. |
| 9 | **ASKfolio** | A *folio* of picture cards — polished, portfolio-like feel. |
| 10 | **ASKprint** | Laser-focused on the print output; immediately communicates purpose. |
| 11 | **ASKspark** | Suggests communication sparking to life; warm and approachable. |
| 12 | **ASKboard** | Evokes communication boards and card sets used in AAC practice. |

## AAC-based names

| # | Name | Rationale |
|---|------|-----------|
| 13 | **AACard** | AAC + card — the double-A reads as the acronym, the name stays pronounceable. |
| 14 | **AACraft** | AAC + craft — the same wordplay as ASKcraft but anchored to the AAC term. |
| 15 | **AACdeck** | AAC + deck — parallel to ASKdeck; works well internationally where AAC is better known. |
| 16 | **AACstudio** | Mirrors ASKstudio with the internationally recognised AAC acronym. |
| 17 | **AACpress** | Professional print-tool feel; AAC is front-and-centre. |
| 18 | **AACsemble** | AAC + assemble — you *assemble* a set of cards. Unique and memorable. |
| 19 | **AACkit** | Parallel to ASKkit; implies a ready-made toolkit for practitioners. |
| 20 | **AACprint** | Simple, searchable, describes the output format at a glance. |
| 21 | **AACfolio** | A polished portfolio of picture cards; appeals to SLPs and educators. |
| 22 | **AACflow** | Smooth end-to-end workflow from image drop to printed PDF. |

## Shortlist recommendation

If forced to pick one, **ASKdeck** strikes the best balance:

- Short (7 characters), memorable, easy to say aloud
- "Deck" implies a *complete, curated set* of cards — which is exactly what the tool produces
- Works as both an app name and a command (`askdeck sessions/…`)
- Pairs nicely with a logo concept (a stack of colourful cards)
